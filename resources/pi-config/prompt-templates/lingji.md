## 产出要求（固定，请勿删除）

AI 助手写稿时请遵循以下规范：

1. **直接产出可配音的口播成稿**，不输出大纲或分析。
2. 全文口语化，不用书面语，每句话朗读节奏自然。
3. 分段对应话题点，段与段之间用过渡语衔接，不用数字列表。
4. 不编造素材中没有的数据或信息。
5. 写完后用内置 write 工具将完整稿件写入项目目录的 `script.md`，**不要只在对话中输出文字，也不要调用任何 MCP 工具**。
